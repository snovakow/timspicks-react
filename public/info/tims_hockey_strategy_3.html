Tims Hockey Challenge Strategy Analysis

Overview

This document captures a full strategic analysis of the Tims Hockey Challenge, including probability modeling, correlation effects, simulation results, and optimal strategies for different goals such as streaks, expected value, and leaderboard performance.

Key Strategic Insights

  • Correlation between picks significantly impacts outcomes.
  • Independent picks maximize streak probability.
  • Stacked picks maximize variance and leaderboard potential.
  • Hybrid (2+1) structure provides best expected value.

Probability Framework

P(≥1 hit) = 1 - (1 - p1)(1 - p2)(1 - p3)

P(3 hits) = p1 × p2 × p3

Variance Insights

Lower probability environments (e.g., 40% / 25% / 15%) increase variance and make correlation more valuable.

Final Conclusions

  • Use independent picks for streaks.
  • Use hybrid strategy for consistent point accumulation.
  • Use stacking for leaderboard wins.
  • Avoid opposing team picks due to negative correlation.
